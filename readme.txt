Displays readability level for your posts and pages.

== Description ==

Simply install. Readability level for your posts and pages will appear on the appropriate table.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/readability` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

That's it!
